## :rocket: Getting Started

First, install all dependencies:

```bash
npm install
# or
yarn
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
```


Runs the app in the development mode.\
Open [http://localhost:3333](http://localhost:3333) to view it in the browser.
